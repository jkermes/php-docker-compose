# Dead simple LAMP stack with Docker compose

## Installation
Create environment file from template
```bash
cp env.template .env
```

The following variables can be adjusted

| Name                | Default value |
| ------------------- | ------------- |
| MYSQL_ROOT_PASSWORD | passwd        |
| MYSQL_USER          | user          |
| MYSQL_PASSWORD      | passwd        |
| MYSQL_DATABASE      | mydb          |
